Navegação lateral fixa de altura total

Tente rolar esta área e veja como o sidenav adere à página

Observe que este elemento div tem uma margem esquerda de 25%. Isso ocorre porque a navegação lateral está definida para 25% de largura. Se você remover a margem, o sidenav irá sobrepor/ficar em cima deste div.

Observe também que definimos overflow:auto para sidenav. Isso adicionará uma barra de rolagem quando o sidenav for muito longo (por exemplo, se tiver mais de 50 links dentro dele).
