Exemplo de Sidenav responsivo

Este exemplo usa consultas de mídia para transformar anavegação lateral em uma barra de navegação superior quando o tamanho da tela é de 900px ou menos.

Também adicionamos uma consulta de mídia para telas de 400px ou menos, que empelhará e centralizará verticalmente os links de navegação.

Redimensione a janela do navegador para ver o efeito.
